# Apexify Technologies AI Projects

Welcome to the **Apexify Technologies AI Internship** project repository! This repository contains a suite of four distinct Artificial Intelligence and Machine Learning applications, all of which have been fully upgraded with **modern, premium web interfaces** and **REST API backends**.

These projects showcase practical implementations of NLP, Computer Vision, Deep Learning, and AI decision-making.

---

## Projects Overview

### 🎮 [Task 1: Rock-Paper-Scissors AI](./Task1_RockPaperScissors_AI)
A dynamic web game where you play against a smart AI opponent.
- **Key Technology:** Pattern Recognition, Flask, HTML/CSS/JS.
- **Highlights:** The AI analyzes your past move history to predict your next action and strategically counters it. Features a premium glassmorphism UI and fluid animations.

### 💬 [Task 2: FAQ Chatbot](./Task2_FAQ_Chatbot)
An intelligent IT Assistant chatbot designed to handle common technical queries.
- **Key Technology:** Natural Language Processing (NLP), NLTK, Scikit-Learn (TF-IDF, Cosine Similarity).
- **Highlights:** Understands intents (Greetings, Farewells) and accurately matches user queries against a custom JSON knowledge base. Features a sleek, messenger-style web interface.

### 😃 [Task 3: Emotion Image Classifier](./Task3_Image_Classifier)
A real-time webcam application that detects faces and classifies their emotions.
- **Key Technology:** DeepFace, Keras/TensorFlow, OpenCV, Flask.
- **Highlights:** Streams webcam frames from the browser to the backend, returning real-time bounding boxes and emotion labels (Happy, Sad, Neutral, etc.) overlaid on a dynamic HTML5 canvas.

### 🎯 [Task 4: Object Detection & Tracking](./Task4_Object_Detection_Tracking)
A high-performance object detection and tracking system using live webcam feeds.
- **Key Technology:** YOLOv8 (`ultralytics`), PyTorch, BoT-SORT algorithm.
- **Highlights:** Upgraded to use the **Open Images V7** dataset model (`yolov8n-oiv7.pt`), capable of identifying over 600 distinct objects. Features persistent object ID tracking and a cyberpunk-themed HUD displaying active object counts and FPS.

## General Setup

Each project is designed to run independently with its own backend server. To run any project:
1. Navigate into the specific project folder.
2. Install its dependencies: `pip install -r requirements.txt`
3. Run the Flask application: `python app.py`
4. Open the local address printed in the terminal (e.g., `http://127.0.0.1:5000`) in your web browser.
